# Imagen Requerida: paes-scatter-plot.png

Para que la publicación funcione correctamente, necesitas agregar la imagen del scatter plot de PAES.

## Ubicación

Coloca la imagen en:
```
images/paes-scatter-plot.png
```

## Descripción de la Imagen

La imagen debe ser el gráfico de dispersión que muestra:
- Fondo negro
- 4 clusters de puntos de colores (rojo, azul, amarillo, verde)
- Líneas horizontales y verticales punteadas dividiendo el gráfico en 4 cuadrantes
- Ejes que van aproximadamente de 400 a 850

## Formato Recomendado

- **Formato**: PNG
- **Resolución**: Mínimo 1200px de ancho
- **Tamaño**: Optimizado para web (máximo 500KB)

## Después de Agregar la Imagen

Una vez que agregues la imagen, haz commit y push:

```bash
git add images/paes-scatter-plot.png
git commit -m "Agregar imagen de scatter plot PAES"
git push
```
